Projects

Hardware

A.L.U.(Arithmetic Logic Unit)

An arithmetic logic unit (ALU) is a digital circuit used to perform arithmetic and logic operations. It represents the fundamental building block of the central processing unit (CPU) of a computer. Designed a 4-bit Arithmetic Logic Unit as a part of a group project with some of my friends. Learnt much about hardware handling and issues while debugging.

Automated Home Management System

In this project we tried to show a prototype how we can make our home automated using micro controller and some other controlling devices(HC-05 Blue-tooth module,Digital TCRT-500 IR Sensor Array04). This project will also help to reduce the wastage of energy. The lights and fans of a room will be automatically on if anyone enters the room. If there is no one in the room and one of the lights and fans are on,it will automatically turn off. We also provided options for turning on/off the lights and fans using android phone via bluetooth module.

Software

Clock-Calendar

It's a simple calendar and clock project in C, with the help of igraphics. User can get calendar of any month of any year providing a valid input. Also the local time will be shown in digital clock display.

Ludo(BOX & Snake)

Ludo Game(Both Box Ludo and Snake Ludo) in C. in Box Ludo 4 Players can play in turns as per convention and in snake ludo 2 players can play. A main menu appears at the beginning to choose the type of game.

Multi-Player Monopoly Game

A Monopoly Game in Java( Usage of Threads & Network utilities of Java). 4 players can play it as per convention if connected to the same network. Game starts upon calling of the server and then players enter by calling client file.
